**ABOUT**

Combines multiple carts so that users only ever have one.

**DEPENDENCIES**

- commerce_cart (from Drupal Commerce 2.x)

**INSTALLATION**

Enable the module. 

**USAGE**

It works automatically as carts are assigned to users, and and as users log in.
